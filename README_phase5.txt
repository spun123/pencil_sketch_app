Этап 5 проекта

Что добавлено:
- Режим экономии памяти для локальных AI-режимов
- Автоматические повторные попытки генерации при нехватке памяти
- Понижение разрешения и шагов генерации при fallback
- Подготовка к сборке в exe

Как работает fallback:
- первая попытка идет в обычном режиме
- если памяти не хватает, программа сама уменьшает нагрузку
- если включен флажок "Экономия памяти", щадящий режим включается сразу

Что добавлено для сборки:
- build_exe.bat
- requirements_build.txt

Как собрать exe:
1. Установить обычные зависимости:
   pip install -r requirements.txt
2. Установить сборщик:
   pip install -r requirements_build.txt
3. Запустить build_exe.bat

Важно:
- Локальные режимы InstantID и IP-Adapter по-прежнему требуют модели в папке models
- Экономия памяти помогает, но не заменяет видеокарту полностью
